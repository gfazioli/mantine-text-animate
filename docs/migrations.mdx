## Upgrade guide

This page collects the **breaking changes** and **migration steps** for each major version of `@gfazioli/mantine-text-animate`. If you are starting a new project, you can skip this page ‚Äî everything here is about updating existing code from an older version.

New features introduced in a release (that do not break anything) are **not** listed here. See the [GitHub Releases](https://github.com/gfazioli/mantine-text-animate/releases) page or the newsletter for the full changelog.

## v3

v3.0.0 introduces new features and breaking changes:

### Breaking changes in v3

- **Spinner `children` type**: Now accepts `string | React.ReactNode[]`. When passing a `ReactNode[]`, text processing (repeat/reverse) is skipped and you must provide an explicit `aria-label`.
- **`speed` prop semantics**: The `speed` prop on Typewriter, Gradient, and Highlight now works as a multiplier ‚Äî higher values mean faster animation (consistent with all other components). Previous behavior was inverted (higher = slower). Default values changed: Typewriter `0.03` ‚Üí `1`, Gradient `3` ‚Üí `1`. If you were using custom `speed` values, invert them (e.g., old `speed={0.01}` on Typewriter ‚âà new `speed={3}`).

### New features in v3

- **`onCharType` callback** on Typewriter ‚Äî called for each character typed with `(char, index)`
- **`pauseAt` prop** on Typewriter ‚Äî map of char index ‚Üí pause duration (ms) for custom pauses
- **`prefix` / `suffix` props** on NumberTicker ‚Äî render content before/after the number
- **`onAnimationComplete` callback** on TextAnimate ‚Äî fires when all segments finish animating
- **`trigger` prop** on TextAnimate ‚Äî `'mount'` (default), `'inView'` (IntersectionObserver), or `'manual'`
- **`useTextAnimate()` hook** ‚Äî returns `{ animate, setAnimate, replay, isAnimating, key, onAnimationComplete }`
- **`TextAnimate.Gradient`** compound component ‚Äî animated gradient text via `background-clip: text`
- **`animate="loop"`** on TextAnimate ‚Äî automatic in‚Üípause‚Üíout‚Üípause cycle with configurable `loopDelay`
- **`withSound` prop** on Typewriter ‚Äî synthesized mechanical keyboard click via Web Audio API
- **`formatValue` prop** on NumberTicker ‚Äî override the default `Intl.NumberFormat` formatting
- **Scramble mode** on TextTicker ‚Äî deterministic per-character scramble via `scrambleDuration` and `staggerDelay`
- **`TextAnimate.Highlight`** compound component ‚Äî animated highlighter marker effect (CSS-only)
- **`TextAnimate.SplitFlap`** compound component ‚Äî airport departure board (Solari board) 3D flip display
- **`TextAnimate.Morphing`** compound component ‚Äî fluid text transitions using LCS algorithm
- **`TextAnimate.RotatingText`** compound component ‚Äî animated text carousel with slide/fade/blur transitions
